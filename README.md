# ART Brother Site v5 PWA Final

Versão final do site da ART Brother com recursos de aplicativo instalável (PWA).

Inclui:
- Site institucional completo
- Menu mobile
- Carrossel automático
- Galeria com lightbox
- Botões WhatsApp e Instagram
- Botão Solicitar Orçamento fixo
- Botão Compartilhar
- Botão Favoritar
- Botão Instalar App (PWA)
- Service Worker para cache básico
- Manifest PWA
- Favicons com AB legível
- Schema.org LocalBusiness
- sitemap.xml
- robots.txt
- Página 404 personalizada

Para funcionar como app instalável, publique o site em HTTPS.
