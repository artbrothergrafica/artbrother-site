# ART Brother Site Final

Versão final com menu mobile, glassmorphism, carrossel, lightbox, botões sociais, compartilhamento, favorito, botão fixo de orçamento, SEO local com Schema.org, página 404, sitemap, robots e favicons otimizados.

Para publicar, envie todos os arquivos para a raiz do repositório/site. Mantenha a pasta `favicon/` junto aos arquivos principais.

Observação: mantenha também a pasta `img/` original do projeto com banners, produtos e galeria.
