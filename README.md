# ART Brother Site v5 PWA Final

Versão final do site da ART Brother com recursos de aplicativo instalável (PWA).

Inclui:
- Site institucional completo
- Menu mobile
- Carrossel automático
- Galeria com lightbox
- Botões WhatsApp e Instagram
- Botão Solicitar Orçamento fixo
- Botão Compartilhar
- Botão Favoritar
- Botão Instalar App (PWA)
- Service Worker para cache básico
- Manifest PWA
- Favicons com AB legível
- Schema.org LocalBusiness
- sitemap.xml
- robots.txt
- Página 404 personalizada

Para funcionar como app instalável, publique o site em HTTPS.


## Atualização v5.1 definitiva

Esta versão usa o ícone oficial da ART Brother em todos os favicons e ícones do PWA.

Arquivos principais de ícone:
- favicon/favicon-16x16.png
- favicon/favicon-32x32.png
- favicon/favicon-48x48.png
- favicon/favicon-64x64.png
- favicon/favicon-180x180.png
- favicon/favicon-192x192.png
- favicon/favicon-512x512.png
- favicon/apple-touch-icon.png
- favicon/android-chrome-192x192.png
- favicon/android-chrome-512x512.png
- favicon/favicon.ico

O cache do PWA foi alterado para `art-brother-v5.1.0-oficial`.

Se o navegador ou aplicativo instalado ainda mostrar o ícone antigo:
1. Desinstale o app antigo.
2. Limpe os dados/cache do site no navegador.
3. Acesse o site novamente em HTTPS.
4. Instale o app novamente.
